# Reservas API

Mini API para gestionar reservas, desarrollada como parte del curso de Metodologías de Desarrollo.

## Objetivo del proyecto
Construir una API REST básica utilizando FastAPI y aplicar buenas prácticas de desarrollo como uso de Git, documentación y organización del proyecto.

## Funcionalidades
- Listar reservas.
- Crear reservas.
- Confirmar reservas.

## Tecnologías utilizadas
- Python
- FastAPI
- Uvicorn
- Git
- GitHub
